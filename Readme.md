# npm install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save
```

# Getting Started with IPL Project

## Available Scripts

In the project directory, you can run:

```sh
$ npm start
```

# Import Lodash Library in project

```sh
  $ npm i lodash
```

# Import http module in project

```sh
$ npm i http
```

# Install nodemon for server

```sh
   $ npm i nodemon
```
 
# Run Nodejs Server 
 ```sh
   npm run server
 ```
